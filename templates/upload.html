Filling Out the File Details
  • Title: Short, descriptive filename (e.g., "Jane Doe at Toronto Film Festival 2024")
  • Caption: A brief one-line summary (e.g., "Jane Doe during a photocall at the 2024 Toronto Film Festival")
  • Description: Defaults to caption, but you can use the description to add more context if helpful‚Äîwhere, when, why the photo was taken
  • Date: Usually auto-filled from your photo's metadata

Adding Categories

Categories help organize photos on Commons like folders in a filing system. The WikiPortraits category should already be added if you used the upload links above.

Also add at least one of these:

  • Subject's full name (for notable people)
  • Event name (e.g., "Toronto International Film Festival 2024")
  • Location (e.g., "Gladstone Hotel" or "Toronto")

Tip: Start typing and suggestions will appear. Don't worry if you're unsure‚Äîexperienced editors can help refine categories later.
